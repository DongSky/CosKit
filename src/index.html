快速开始

  1. 点击「+ 新建会话」或拖拽图片上传一张照片
  2. 在底部输入框输入修图指令（如"美白磨皮"、"换个二次元背景"）
  3. 等待 AI 返回修图结果，点击缩略图可放大查看
  4. 继续输入指令可基于当前结果迭代编辑，点击历史节点可从任意节点分支

方法一：设置面板配置 API

点击右上角 ⚙ 设置按钮 → 选择「API 配置」tab → 填写 API Key、Base URL 和超时时间 → 保存即可。

方法二：使用 .env 文件

在程序同级目录下创建 .env 文件，内容如下：

GEMINI_API_KEY=your_api_key_here
GEMINI_BASE_URL=https://your-proxy-url.com/v1beta/models/gemini-3.1-pro-preview:generateContent
GEMINI_IMAGE_BASE_URL=https://your-proxy-url.com/v1beta/models/gemini-3-pro-image-preview:generateContent

支持的环境变量

  • GEMINI_API_KEY — Gemini API 密钥
  • GEMINI_BASE_URL — 文本模型代理地址（含模型名）
  • GEMINI_IMAGE_BASE_URL — 图像模型代理地址（含模型名）

设置面板中的配置优先级高于 .env 文件。留空则自动使用 .env 中的值。
